This folder is used to store run configurations for JetBrains IDEs (like WebStorm), which can be shared across team members. Each configuration is stored as a separate XML file. These are mostly Debug Javascript configurations pointing to various plunkers.

If you wish to add your own run configuration, you can do so by creating a new run configuration in your IDE and clicking 'Store as a project file' option. This will prompt you to enter the folder path, just use the default value.
