# 三端打包与抽测表（L4）

| 平台 | 产物形态 | 构建 | 本仓状态 | 安装抽测 |
| --- | --- | --- | --- | --- |
| Windows | 便携 `.exe` + `.msi` | `cd desktop && npm run tauri build` | **已出**：`dist/windows/` | ☐ 你验收双击 exe |
| macOS | `.dmg` / `.app` | CI `aos-desktop-build.yml` | 由 Actions 矩阵产出 | ☐ |
| Linux | AppImage / deb | 同上 | 由 Actions 矩阵产出 | ☐ |

抽测：安装或运行后窗口标题「AOS 本地工作站」；健康灯可刷新；adapter+Dify（或 `BUDDY_DEMO_MODE=1`）可问答。
